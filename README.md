# Inbank Decision Engine

This project implements a loan decision engine as part of the Inbank internship assignment.

## Tech stack

- Backend: Java + Spring Boot
- Frontend: Next.js (React + TypeScript)

## Structure

- /backend — REST API with decision logic
- /frontend — UI for interacting with the decision engine

## How to run

### Backend
cd backend
./mvnw spring-boot:run

### Frontend
cd frontend
npm run dev
